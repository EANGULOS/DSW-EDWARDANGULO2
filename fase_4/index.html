Hojas de Estilo en cascada css

Css es el formato recomendado para las paginas escritas en formato HTML en base alos estandares
Cascading Style Sheets(hojas de estilo en cascada), publicado por el Word Wide Web Consortium(W3C). El uso de este estandar nos va ahorrar tiempo, da consistencia y facilita en gran medida la escritura de paginas Web.Vamos a aprender a utilizar las hojas de estilo cascada, para crear pagina Web mas flexibles y ligeras, controlando su aspecto grafico con mayor presicion y con mayor facilidad para la correcion de errores.
Sintaxis de Css: La sintaxis consta de dos partes: un selector y una declaracion.
Ejemplo: h1 {color:green} Donde h1 es selector y color:green la declaracion.
Existen dos tipos de selectores
Selectores de grupo:Si tiene que aplicar estilos a mas de un elemento puede agrupar los selectores que compartan las mismas definiciones.Para ello se escriben uno tras otro separados por una coma.
En el siguiente ejemplo h1 y los elementos hasta el h6 comparten la misma definicion.
h1,h2,h3,h4,h5,h6
{color:green}
Selectores de clase:Permite establecer diferentes estilos que se pueden aplicar al mismo elemento.Imagine que usted necesita tener dos parrafos diferentes en el documento uno alineado a la derecha y el otro al centro.Es asi como estos selectores de clase hacen que esta tarea sea mas facil

Durango, A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.

Frameworks CSS

Un framework de CSS es una biblioteca de estilos que puede ser usada para implementar diseños web. Aportan una serie de utilidades que pueden ser aprovechadas frecuentemente en los distintos diseños web.
Los Frameworks de Css sde clasifican en dos tipos: Multiproposito:Proporciona un conjunto de utilidades variadas utiles para distintos aspectos del desarrollo de un diseño Web. Ejemplo:Cascade framework
Proposito especifico:Estan orientados a proporcionar herramientas para un proposito determinado por ejemplo:hay framework de Css orientados a proporcionar herrramientas para el desarrollo de rejillas (Ej. Responsive Grid System, Skeleton, Semantic Grid System), orientadas al control de la parte gráfica de la interfaz de usuario (Ej. Accordion, tabs, slideshow), orientados a la tipografía, etc.
